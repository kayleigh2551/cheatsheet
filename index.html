HTML tags cheatsheet

Tag Name Description
<p> Plain text Contains a block of plain text
<span> Span Contains short pieces of text or other HTML and is used to seperate content on the same line
<em> Emphasis Emphasises text in italics
<strong> Strong Emphasises text in bold
<break> Line break Creates a line break
<ul> Unordered list Creates an unordered list
<ol> Ordered list Creates an ordered list
<li> Line items Adds line items to an unordered or ordered list
<image> Image Adds image, use with src=''. Doesn't need the normal closing tag'
<video> Video Adds video, use with src='' and controls for height and width. Needs closing tag
